## 项目 / Projects

### [Unturned Announcement UI Plugin](https://github.com/ouxiaaocha/Unturned-Announcement-UI-Plugin)

Unturned 服务器公告 UI 插件，支持规则展示、轮播公告、临时强制广播与聊天同步。  
An announcement UI plugin for Unturned servers with rules, rotating messages, forced broadcasts, and chat sync.

### [Unturned Infinite Reload Ammo](https://github.com/ouxiaaocha/Unturned-Infinite-Reload-Ammo)

换弹式无限子弹插件，保留原版换弹动作，并在换弹后自动补满弹匣。  
A reload-based infinite ammo plugin that keeps the original reload flow and refills magazines automatically.

### [Unturned CBI Plugin](https://github.com/ouxiaaocha/Unturned-CBI-Plugin)

Unturned 虚拟云存储插件，面向多保险库、权限组和自动保存场景。  
A virtual cloud storage plugin for Unturned servers with vaults, permission groups, and auto-save.
